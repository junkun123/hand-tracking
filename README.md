# Page

## 🎯 Hand Tracking

Detección y seguimiento de manos en tiempo real usando IA.

### ¿Qué es?

Hand Tracking es un programa que detecta y rastrea tus manos en tiempo real usando tu cámara web. Utiliza MediaPipe de Google para un reconocimiento preciso de 21 puntos clave por mano.

### Características principales

* ✅ Detección de hasta 2 manos simultáneamente
* ✅ Rastrea 21 puntos de referencia por mano
* ✅ Funciona en tiempo real sin lag
* ✅ Compatible con Windows, Mac y Linux
* ✅ Instalación automática con un comando

### Inicio rápido

```bash
git clone https://github.com/junkun123/hand-tracking.git
cd hand-tracking
python setup.py
python hand_tracking.py
```
